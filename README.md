# 9Router - AI Router Gateway

Connect 60+ AI providers through one OpenAI-compatible endpoint.

## Features

- 🔄 Smart 3-tier fallback (Subscription → Cheap → FREE)
- 🚀 RTK Token Saver (save 20-40% tokens)
- 📊 Real-time quota tracking
- 🎨 40+ AI providers support
- 🔐 OAuth + API key management

## Usage

1. Connect providers in dashboard
2. Point your CLI tool to `http://localhost:20128/v1`
3. Use models via combos with auto-fallback
